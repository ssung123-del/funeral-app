# 🕊️ 장례문자 생성기

한국 교회용 장례 안내 문구를 빠르게 생성하는 단일 파일 웹앱입니다.  
`교구장`/`교회장` 모드를 지원하고, Step 1 결과를 기반으로 Step 2 교우소식(상세)까지 바로 작성할 수 있습니다.

## ✨ 주요 기능

### Step 1 — 장례 문자 생성
- `교구장 모드`
  - 일반 위로예배/개별조문(발인 날짜만) 지원
  - 관계 뒤 `소천` 자동 정규화(중복 입력 방지)
  - 90바이트 초과 시 원본(LMS) + 추천 단축안 자동 제시
    - 1차: `교회출` → `교출`
    - 2차: 여전히 초과 시 `소천` 제거
- `교회장 모드`
  - `겹장(합동 장례)` 지원 (교구장/교회장 혼합 가능)
  - 위로/입관/천국환송 일정 중 입력된 항목만 출력
  - `故` 자동 접두
- 스마트 시간 포맷
  - 시간 3개를 그룹으로 보고 하나라도 분(`:MM`)이 있으면 정시도 `:00`으로 통일
  - 예: `2시 / 3:30 / 6시` → `오후2:00 / 오후3:30 / 오전6:00`

### Step 2 — 교우소식(상세) 작성
- Step 1 데이터를 자동 복사해 상세 문구 생성
- 교회출발 체크 ON/OFF 반영
- `빈소`는 Step 1 `장례식장` 값이 자동 반영됨 (별도 입력 없음)
- `장지`만 추가 입력

## 🚀 사용 방법

1. `index.html`을 브라우저에서 열기
2. 모드 선택 (`교구장` 또는 `교회장`)
3. 고인 정보/일정/장례식장 입력
4. `문자 생성하기` 클릭 후 결과 복사
5. 필요 시 `교우소식(상세) 작성하기`로 Step 2 생성

## 📋 생성 예시

### 교구장
```text
[교구장-강남]홍길동집사(김영희권사)부친소천/위로:15일(오늘)오후2시교회출/OO병원장례식장3호
```

### 교회장
```text
[교회장-강남]故홍길동장로(김영희권사)소천/위로:15일(토)오후4시교회출/입관:16일(일)오후2시교회출/천국환송:17일(월)오전6시교회출/OO병원장례식장3호
```

### 교우소식(상세)
```text
[교회장-강남] 故홍길동장로(김영희권사) 소천

위로: 2월 15일(토) 오후 4시 교회출발
입관: 2월 16일(일) 오후 2시 교회출발
천국환송: 2월 17일(월) 오전 6시 교회출발
빈소: OO병원장례식장3호
장지: OO공원묘지
```
