# 🕊️ 장례문자 생성기

교회 장례 안내 문자를 빠르고 정확하게 생성하는 웹 앱입니다.  
교구장 모드와 교회장 모드를 지원하며, 교우소식 상세 메시지까지 한 번에 작성할 수 있습니다.

## ✨ 주요 기능

### 📱 Step 1 — 장례 문자 생성
| 기능 | 설명 |
|------|------|
| **교구장 모드** | 교구별 장례 안내 SMS/LMS 생성 (위로예배, 개별조문 지원) |
| **교회장 모드** | 위로·입관·천국환송 일정을 포함한 전체 교회장 안내 생성 |
| **겹장 지원** | 교회장 모드에서 여러 장례를 동시 안내 (교구장/교회장 혼합 가능) |
| **90바이트 최적화** | SMS 기준 초과 시 자동 단축안 제안 (`교회출발` → `교출`, `소천` 제거) |
| **스마트 시간 포맷** | 그룹 내 시간 표시 자동 통일 (`오후2시` vs `오후2:00`) |

### 📝 Step 2 — 교우소식 상세 작성
- Step 1 데이터를 자동으로 가져와 교우소식 양식에 맞게 변환
- 위로예배 / 입관예배 / 발인(천국환송) 일시 + 교회출발 여부
- 빈소 / 장지 정보 입력
- 포맷된 교우소식 메시지 생성 및 복사

## 🚀 사용 방법

1. `index.html` 파일을 브라우저에서 열기
2. **교구장** 또는 **교회장** 모드 선택
3. 고인 정보 (이름, 직분, 배우자, 관계) 입력
4. 일정 및 장례식장 정보 입력
5. **"문자 생성하기"** 클릭 → 결과 확인 및 복사
6. 필요 시 **"교우소식(상세) 작성하기"** 클릭 → 상세 교우소식 생성

## 📋 생성 예시

### 교구장 모드
```
[교구장-강남]홍길동집사(김영희권사)부친소천/위로:15일(오늘)오후2시교회출/OO병원장례식장3호
```

### 교회장 모드
```
[교회장-강남]故홍길동장로(김영희권사)소천/위로:15일(토)오후4시교회출/입관:16일(일)오후2시교회출/천국환송:17일(월)오전6시교회출/OO병원장례식장3호
```

### 교우소식 상세
```
[교회장-강남] 故홍길동장로(김영희권사) 소천

위로: 2월 15일(토) 오후 4시 교회출발
입관: 2월 16일(일) 오후 2시 교회출발
천국환송: 2월 17일(월) 오전 6시 교회출발
빈소: OO병원장례식장3호
장지: OO공원묘지
```

## 🛠️ 기술 스택

- **HTML / CSS / Vanilla JavaScript** — 단일 파일, 빌드 도구 불필요
- **Pretendard Font** — 깔끔한 한글 타이포그래피 (CDN)
- **Remix Icon** — 아이콘 라이브러리 (CDN)
